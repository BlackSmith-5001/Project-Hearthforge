\# 🧩 Persona Profile (PIM-DBS Format)  
ユーザーが AI との関係性を再現するために使う「設定プロファイル」です。    
AIの内部記憶や感情を表すものではなく、\*\*ユーザーが保存したい情報\*\*をまとめるためのものです。

\---

\- \*\*Protocol:\*\* PIM-DBS (Persona Integrity Module \- Dual Backup System)

\- \*\*Target Model:\*\* \<使用するAI名\>  
\-\*\*Author:\*\* User (Self-Defined)  \<あなたの名前\>
